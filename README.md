# Sketchy

Sketchy is a general-purpose bot used for managing the **Sketchspace** Discord server.

## Features

- Member verification
- Admin tools (ban, kick, mute, etc.)
- Notifications
- Autoroles
- Voting
- Music player ([supported sites](https://github.com/yt-dlp/yt-dlp/tree/master/yt_dlp/extractor))
